核心设计原则

  1. 副作用注册：工具通过 import 副作用自动注册到全局注册表，无需显式初始化
  2. 提示缓存优先：System prompt 整个会话保持稳定，确保 prefix cache 命中
  3. 渐进披露：技能/记忆按需加载，避免 token 浪费
  4. 平台无关性：通过依赖注入回调，Agent 核心不感知运行平台
  5. 冻结快照：记忆写入即持久化，但当前会话 prompt 不变，下次会话生效
